# Font Resources for PDF Generation

This directory contains TTF font files used for generating localized invoices.

## Required Fonts

Place the following font files in this directory:

- `Cairo-Regular.ttf` - For Arabic and Latin text
- `Cairo-Bold.ttf` - For bold Arabic and Latin text

OR

- `Amiri-Regular.ttf` - Alternative Arabic font
- `Amiri-Bold.ttf` - Alternative bold Arabic font

## Why These Fonts?

These fonts support both Arabic and Latin scripts, ensuring invoices render correctly in both English and Arabic.

## Download Sources

- Cairo: https://fonts.google.com/specimen/Cairo
- Amiri: https://fonts.google.com/specimen/Amiri

The fonts will be embedded into generated PDFs to ensure consistent rendering across all devices.
